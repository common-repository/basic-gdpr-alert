== Description ==

Basic GDPR Alert, adding GDPR alert to your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/basic-gdpr-alert directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Appearance->Customize->Basic GDPR Alert screen to configure the plugin.
